HTML

Stands for "Hypertext Markup Language." HTML is the language used to create webpages. "Hypertext" refers to the hyperlinks that an HTML page may contain. "Markup language" refers to the way tags are used to define the page layout and elements within the page.

CSS

Stands for "Cascading Style Sheet." Cascading style sheets are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of web pages that previously could only be defined in a page's HTML. CSS can be used seperately in a .css file or inside a HTML file.

JavaScript

JavaScript is a programming language used on the client side to make the web page more interactive. It can run on the web page without having to contact the server.
